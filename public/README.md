# Public Assets

## PWA Icons

You need to create and add the following icon files for PWA support:

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

### Creating Icons

You can use any image editor or online tool to create these icons. The icons should:
- Feature the El Haj'Aime logo or barbershop theme
- Use the brand colors (navy #1e3a8a, gold #d4af37)
- Be square format
- Have transparent or solid background

### Recommended Tools

- [Favicon.io](https://favicon.io) - Generate icons from text/emoji
- [RealFaviconGenerator](https://realfavicongenerator.net) - Comprehensive favicon generator
- Adobe Photoshop/Illustrator
- Canva

### Quick Solution

If you need placeholder icons quickly, you can:
1. Create a simple square image with "EH" text in gold on navy background
2. Export as PNG at both sizes
3. Place in this directory

The app will work without these icons, but PWA installation won't be fully functional.
